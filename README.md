SALES PREDICTION OF RETAIL STORE

📌 Project Overview
This project predicts future retail sales using Machine Learning. It analyzes historical sales data and business-related features such as promotions, holidays, and store type to forecast sales. The model helps retailers make informed decisions about inventory, staffing, and marketing strategies.

🎯 Objective
Build a regression model that accurately predicts future sales based on historical data and store-related features.

📂 Dataset
Dataset: Rossmann Store Sales (Kaggle)

Features include:
Store
Sales (Target Variable)
Customers
Open
Promotion
State Holiday
School Holiday
Store Type
Date

🛠️ Technologies Used
Python
Jupyter Notebook
Pandas
NumPy
Matplotlib
Scikit-learn

📊 Project Workflow
Import Libraries
Load Dataset
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Feature Encoding
Train-Test Split
Train Linear Regression Model
Predict Sales
Evaluate Model Performance
Save Prediction Results to CSV

🤖 Machine Learning Model
Linear Regression

📈 Model Evaluation
The model is evaluated using:
Mean Squared Error (MSE)
Root Mean Squared Error (RMSE)
R² Score

📁 Output
The project generates:
Predicted Sales values
Sales_Prediction_Results.csv containing Actual vs Predicted Sales
